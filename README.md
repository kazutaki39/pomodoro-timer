# Pomodoro Focus Timer

シンプルなポモドーロタイマーです。ブラウザで動作し、インストール不要です。

## 機能

- 作業・短休憩・長休憩の3モード切り替え
- 円形プログレスリングでの残り時間表示
- 4ポモドーロごとに自動で長休憩へ切り替え
- 今日の集中時間・ポモドーロ数・連続記録を表示
- セッションログの記録（localStorage に保存）
- ブラウザ通知対応
- タイマー時間のカスタマイズ設定

## 使い方

1. `index.html` をブラウザで開く
2. ▶ ボタンでタイマー開始
3. 終了すると自動で次のモードに切り替わる

## デフォルト設定

| モード | 時間 |
|--------|------|
| 作業   | 25分 |
| 短休憩 | 5分  |
| 長休憩 | 15分 |

設定パネル（⚙ タイマー設定）から変更できます。
